# Cascade

[Carbon motion guidelines](https://www.carbondesignsystem.com/guidelines/motion/overview/)

## 概述

此模式适用于卡片、瓷砖或类似样式的组件。 在页面上主要关注的区域使用此模式，以帮助用户完成他们的旅程或找到页面上最重要的信息。
如果它是页面的次要焦点，则不应在页面上使用它，因为这会分散用户的注意力。
